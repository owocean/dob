# node-dob
the nodejs module for dob

## install
```
$ npm install node-dob
```  
*you can also clone this repo uwu*

## usage
```javascript
const dob = require("node-dob");
var database = dob("./dobfile.dob");
// or
const database = require("node-dob")("./dobfile.dob");
```
